## Running the tests
We are using [`unittest`](https://docs.python.org/3/library/unittest.html) for our unit test framework.

To run all tests:
```commandline
python3 -m unittest discover . "*.py"
```

To run tests in just a single module:
```
python3 -m unittest <filename>
```
